# Mini Agent Workflow Engine (Code Review Mini-Agent)

This is a small FastAPI-based backend that implements a minimal workflow/graph engine.
It allows defining nodes (Python functions), edges, branching (via conditions), simple looping,
and exposes APIs to create/run graphs and query run state.

This submission implements **Option A: Code Review Mini-Agent** with simple rule-based tools.
